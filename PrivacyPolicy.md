# Privacy Policy

**Quick Search - Spotlight** Chrome Extension

_Last Updated: January 2, 2026_

## Overview

Quick Search - Spotlight is a browser extension that provides fast, local search functionality. We are committed to protecting your privacy.

## Data Collection

**We do NOT collect, store, or transmit any personal data.**

All data accessed by this extension remains entirely on your local device.

## Data Access

The extension accesses the following data **locally** to provide search functionality:

| Data Type    | Purpose                       | Storage    |
| ------------ | ----------------------------- | ---------- |
| Open Tabs    | Search and switch tabs        | Not stored |
| Bookmarks    | Search bookmarks              | Not stored |
| History      | Search recent visits          | Not stored |
| Page Content | Current text and hidden links | Not stored |

## Data Storage

- **No personal data is stored** beyond your browsing session
- **User Preferences** (search priority, language) are stored locally using Chrome Storage Sync settings to sync across your signed-in devices.
- **No data is transmitted** to external servers
- **No analytics or tracking** is implemented
- **No third-party services** are used

## Permissions

| Permission  | Why We Need It                      |
| ----------- | ----------------------------------- |
| `tabs`      | To list and search open tabs        |
| `bookmarks` | To search your bookmarks            |
| `history`   | To search your browsing history     |
| `activeTab` | To display the search overlay       |
| `scripting` | To inject the search interface      |
| `storage`   | To save your preferences (settings) |

## Third-Party Libraries

This extension uses [Fuse.js](https://fusejs.io/) for fuzzy search functionality. Fuse.js runs entirely locally and does not transmit any data.

## Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected in the "Last Updated" date above.
